# Ottoman–Venetian Bridge: Digital Humanities Series

A multi-phase digital project exploring cultural, diplomatic, and economic interactions
between the Ottoman Empire and the Republic of Venice (15th–18th centuries).

Each version represents a focused case study using digital humanities methods
such as TEI/XML encoding, archival research, and map-based storytelling.
